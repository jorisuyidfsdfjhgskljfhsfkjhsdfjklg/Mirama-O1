# Mirama-O1 is NOT a finished project im not even near finishing this project please dont download any models yett expecting to be able to use them

Mirama-O1 utilizes servo controlled Fins in the end of the intake AKA. the exhaust.
the top of the model is an 140mm cone with an extrusion 5mm away from the outer rim of the cone wich is meant to hold an cardboard tube wich is later secured with M3 inserts and and two screws. this cardboard tube is later covered with Either carbon, glass fiber, or kevlar sheets, wich are supposed to be permanantly mountd on the cardboard tube with resin, after this there is the body wich holds EDF, electronics intakes, servos etc, the top of the whole housing is covered with M2 screw holes all spaced 5mm appart for you, yes you to design custome mounting solutions for your preffered electronics. im yet to decide if i want to add custome finns for more stability
